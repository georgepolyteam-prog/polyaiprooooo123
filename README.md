# Poly - AI Prediction Market Terminal

## About

Poly is an AI-powered prediction market analyst for Polymarket. Get real-time market analysis, find mispriced opportunities, and make smarter trading decisions.

## Technologies

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Development

```sh
# Install dependencies
npm i

# Start development server
npm run dev
```

## Deployment

Deploy via your preferred hosting platform (Vercel, Netlify, etc.)
